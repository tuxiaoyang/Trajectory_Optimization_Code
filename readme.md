# Aircraft Trajectory Optimization

## Overview
This repository contains a Python implementation of aircraft trajectory optimization with:
- chance-constrained optimization solved by Gurobi,
- result export (CSV/ACMI) and plotting utilities.

## Requirements
- Python 3.10+
- numpy
- scipy
- pandas
- matplotlib
- sympy
- gurobipy (requires a separate Gurobi installation and valid license)

Install dependencies:

```powershell
pip install -r requirements.txt
```

For Gurobi setup and licensing, see https://www.gurobi.com/.

## Quick Start
1. Optional: generate deterministic warm start.

```powershell
python main_gurobi_initial.py
```

2. Run the chance-constrained optimizer.

```powershell
python main_gurobi.py
```

3. Recreate figures from saved CSV outputs.

```powershell
python plot_from_csv.py
```

## File Structure
- `main_gurobi.py`: Main optimization loop.
- `main_gurobi_initial.py`: Deterministic initializer.
- `ininital.py`: Parameter and scenario initialization.
- `tools.py`: Linearization, Jacobians, coordinate transforms, and helper functions.
- `tools_deter.py`: Helper functions for deterministic initialization.
- `output_info.py`: Output organization and export utilities.
- `plot_picture.py`: Plot functions.
- `plot_from_csv.py`: Rebuild plots/ACMI from CSV outputs.

## Outputs
- `result/output_x_{k}.csv`: state vectors at iteration k.
- `result/output_u_{k}.csv`: control vectors at iteration k.
- `result/acmi.acmi`: ACMI export.
- Figures generated by plotting scripts.

## Notes
- This repository is intended as a reproducible project codebase for optimization experiments.
